فرزاد فروزانفر توسعه‌دهنده فول‌استک

توسعه‌دهنده فول‌استک با ۴ سال تجربه در طراحی و پیاده‌سازی سیستم‌های وب با عملکرد بالا، معماری ماژولار و امنیت پیشرفته. تمرکز اصلی‌ام بر توسعه APIهای سریع، صف‌های پیام‌رسانی، پایگاه‌های داده مقیاس‌پذیر و رابط‌های کاربری مدرن است.

رویکردم به توسعه نرم‌افزار مبتنی بر کدنویسی تمیز، مستند، قابل نگهداری و مؤثر است. از طراحی تا استقرار، ترکیب مناسبی از ابزارهای بک‌اند و فرانت‌اند را برای رسیدن به بالاترین کیفیت فنی به‌کار می‌برم — از ساخت سرویس‌های RESTful تا اجرای سیستم‌های Dockerized در محیط‌های لینوکسی.

[ Terminal Mode: stack.skills.farzad ] 

> Backend::
    PHP 8.1+, Laravel 10.x, Redis (Cache & Queue), MySQL (InnoDB, Joins, Indexes)
    High-performance APIs with Swoole & Hyperf (coroutine, event-loop, async tasks)
    Message broker integration via RabbitMQ (AMQP, delayed queues)
    Custom WordPress Plugin Development (Hooks, Filters, Shortcodes)

> Frontend::
    React.js (Hooks, Context, Router), jQuery (classic interactivity), TailwindCSS 3.x
    Responsive UI, Form integration, Axios API binding

> DevOps::
    Docker (NGINX + PHP-FPM + Redis), Docker Compose (multi-service env)
    Local development orchestration, volume mounting, container optimization

> MessageBroker::
    AMQP protocol implementation with RabbitMQ
    Durable queues, Retry patterns, Delayed messages

> Database::
    Relational: MySQL 8 (ACID, Eloquent ORM, Raw queries, Migrations)
    KeyValue: Redis (TTL, Sets, Hashes, Pub/Sub)

> API::
    RESTful architecture with Resource Controllers, Sanctum token auth
    Structured endpoints, standardized HTTP response codes
    OpenAPI documentation (Swagger/Postman collections)

> Queue::
    Redis Queue (lightweight, fast)
    RabbitMQ DurableQueue (priority queue, failover consumer logic)

> CLI::
    Artisan (command creation, queue:work, scheduler)
    Composer (autoloading, PSR-4, dependency management)
    Bash scripting (deployments, backups)
    NPM / pnpm (frontend build tools, Tailwind CLI, Vite/Webpack)

PHP MVC Framework (Open Source)

فریم‌ورک سبک و ماژولار برای پیاده‌سازی الگوی MVC در PHP که از صفر طراحی شده. این پروژه با هدف یادگیری عمیق‌تر معماری MVC توسعه داده شده و از ویژگی‌هایی مانند روتینگ ساده، جداسازی Web/API، اعتبارسنجی درخواست‌ها، امنیت پیش‌فرض (Rate Limiting)، و Docker پشتیبانی می‌کند.

  • PHP 8+
  • OOP
  • Strategy Pattern
  • Singleton
  • PSR
  • Dockerized

گشتوک — راهنمای گردشگری ایران

یک وب‌اپلیکیشن PWA مبتنی بر Next.js و Laravel که اطلاعات گردشگری شهرهای ایران را به‌صورت ساختاریافته، آنلاین و آفلاین، در اختیار کاربران قرار می‌دهد. گشتوک با رابط تعاملی، نقشه‌، دسته‌بندی هوشمند مکان‌ها و قابلیت نصب به‌عنوان اپلیکیشن، تجربه‌ای کامل برای گردشگران داخلی فراهم می‌کند.

  • Next.js
  • PWA
  • Laravel RESTful API
  • SSR
  • Offline Mode
  • Map & Search
